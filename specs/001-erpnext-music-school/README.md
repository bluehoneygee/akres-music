# ERPNext Music School Spec Kit

Spec kit ini adalah paket dokumen implementasi untuk membangun sistem sekolah musik berbasis ERPNext/Frappe.

## Documents

- [spec.md](./spec.md): kebutuhan produk, scope, role, functional requirements, dan MVP definition.
- [plan.md](./plan.md): strategi implementasi, fase pengerjaan, permission, automation, dan risiko.
- [data-model.md](./data-model.md): rancangan DocType, Custom Field, relasi data, dan business rules data.
- [tasks.md](./tasks.md): task breakdown implementasi dari setup sampai QA.
- [quickstart.md](./quickstart.md): langkah awal membuat custom app dan smoke test.
- [research.md](./research.md): catatan keputusan teknis dan tradeoff.
- [checklists.md](./checklists.md): checklist kualitas spec, data model, security, dan MVP.
- [e2e-manual-payloads.md](./e2e-manual-payloads.md): payload manual untuk mencoba flow Next.js + MongoDB.

## Recommended Build Order

1. Baca [spec.md](./spec.md) untuk menyepakati scope MVP.
2. Pakai [data-model.md](./data-model.md) untuk membuat DocType dan Custom Field.
3. Ikuti [plan.md](./plan.md) untuk urutan fase implementasi.
4. Kerjakan checklist di [tasks.md](./tasks.md).
5. Jalankan smoke test di [quickstart.md](./quickstart.md).
6. Gunakan [checklists.md](./checklists.md) sebelum demo MVP.
